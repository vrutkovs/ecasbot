EC AntiSpam bot для мессенджера Telegram будет автоматически определять и блокировать мультимедийные сообщения и ссылки от недавно вступивших в супергруппу участников, а также ряд известных спамботов.

Для работы данного антиспам бота вы должны сначала получить токен API у @BotFather и указать его на одном из шагов данной программы установки. Данный токен будет сохранён и передан при помощи переменных окружения.
